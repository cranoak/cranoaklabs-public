 # udemy jupyter_setup.md

Jupyter lab is very easy to setup

```
Run on windows wsl 
jupyter-lab --no-browser


sudo apt update -y && sudo apt upgrade -y
python --version
#sudo apt install python3 python3-pip # Optional
pip install --upgrade pip
cd jupyter-notebook-lab/
python -m venv jupyter-lab-venv
source jupyter-lab-venv/bin/activate

pip list
pip install jupyterlab 

```
