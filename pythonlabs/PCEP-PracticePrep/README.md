# PCEP Test README.md

The recommended steps for Certified Entry-Level Python Programmer:

- Sign up for the free course and test at [Certified Entry-Level Python Programmer](https://pythoninstitute.org/pcep) and take course
- Set up jupyter lab following the [Python Setup](../PYTHON_SETUP.md)
- Optionally go through [python tutorial](https://docs.python.org/3/tutorial/index.html) on the official python website.
- Make a copy of the jupyter notebooks )and put them into a folder with your name on it.See [DavesPractice folder](./DavesPractice/) as an example.
- Go through [PCEP-PracticePrep](../PCEP-PracticePrep/) folder.
  These are jupyter notebooks designed to focus on one topic.

## Taking the test

Once you are comfortable will the practice tests, schedule the test.

- Optionally take PCEP practice tests in Udemy
- Go through the PCEP-PracticePrep notebooks especially areas you need focus on
- Go through [PCEP-PracticePrep](../PCEP-PracticePrep/) folder.
- Take PCEP practice test **again**
- Review all wrong answers in old tests
- Take test

## Using this repository

The repository contains jupyter notebooks which the code is commented out.  This allows you to emulate the test by answering what the code produces without seeing the anwsers

### Steps

- Make a directory `<your name>`Practice
- Copy all the jupyter notebooks to the new directory
- Rename the files with you name at the beginning.
  This way you know which file you are editing and don't edit to original

*See DavesPractice* as an example
